Informações sobre SIGTAP

SIGTAP (Sistema de Gerenciamento da Tabela de Procedimentos, Medicamentos e OPM do SUS) é a tabela oficial do Ministério da Saúde que contém:

  • Códigos e descrições de procedimentos médicos
  • Valores de referência para procedimentos
  • Classificação por especialidade e complexidade
API SIGTAP: A integração com API oficial do SIGTAP está em desenvolvimento. Por enquanto, utilize o cadastro manual ou importação via CSV.
